# 引言

> 物理学家视角下的金融市场
>
> A book for the physicist-trained autodidact who wants to take markets seriously as a statistical-mechanics system — without pretending they aren't.

---

## 关于这本书

市面上写金融市场的书大致分两类:

1. **主流金融经济学教科书**(Hull、Bodie、Tirole 这一脉):从无套利、理性人、均衡出发,数学工具是优化、动态规划、Itô 微积分。如果你想做衍生品定价或考 CFA,这是正路。
2. **量化交易实战书**(各类 "alpha cookbook"):从信号、回测、策略组合讲起,假设你已经接受了"市场上有规律,我去抓"的实用主义立场。

**这本书的位置**:写给已经受过物理学训练、对统计力学 / 随机过程 / 临界现象有本科到研究生入门概念的自学者。每个金融概念都从你已知的物理直觉出发,然后讲清楚市场为什么不完全是物理系统、又在哪些地方惊人地像。

**读完之后你能**:

- 用"统计力学系统"的视角理解为什么收益率分布是重尾的、为什么波动率会聚集、为什么协方差矩阵大部分是噪声
- 从经验规律(power law、square-root impact law、log-periodic precursor)出发,识别背后的机制候选
- 跑通基础的 Python lab:画出 S&P 500 的重尾、用 Marchenko-Pastur 给协方差矩阵去噪、跑一个 minority game
- 给同事讲清楚"经济物理学家和主流金融经济学家到底在吵什么",而不是只会站队

**读完之后你不会**:

- 成为衍生品定价专家 —— Black–Scholes 这一脉这本书只会简略提及,不展开
- 拿到一套可以直接上线的交易策略 —— 这不是 alpha cookbook
- 精通测度论级别的随机分析 —— Itô 公式、测度变换够用即可,不追求严谨

## 读者画像

你应该至少满足以下条件:

- 物理 / 应用数学 / 工程 / 计算机科学的本科或以上
- 熟悉 Python,能跑 numpy / scipy / pandas / matplotlib
- 大学微积分、线性代数、初等概率论
- 至少听说过 Brownian motion、power law、Ising model 中的两个

如果你完全没有上述背景,建议先读 Mantegna & Stanley 的导论章,再回来。

## 全书结构

全书 8 个模块,建议**顺序阅读**,但模块 4–7 可以按兴趣交叉读。

| 模块 | 主题 | 关键词 |
|---|---|---|
| **1** 学科诞生 | 物理学家为何进入金融 | Bachelier、Mandelbrot、Stanley、Bouchaud、Sornette |
| **2** 重尾分布 | 高斯不够用,那用什么 | CLT 失效、power law、Lévy stable |
| **3** 随机过程 | 从 GBM 到更复杂的市场动力学 | 跳跃过程、长程相关、波动率聚集 |
| **4** 关联结构与 RMT | 多资产世界里的信号和噪声 | 协方差矩阵、Marchenko-Pastur、去噪 |
| **5** 临界现象 | 相变、Ising、崩盘预警 | 序参量、log-periodic、Sornette |
| **6** 微观结构与订单簿 | 价格是怎么生成的 | 限价单簿、square-root impact、做市 |
| **7** ABM 与涌现 | 从异质 agent 到宏观规律 | minority game、heterogeneous agents |
| **8** 流体力学视角与开放方向 | 把市场写成偏微分方程 | 守恒律、本构关系、反向扩散 |

**模块 8 是物理直觉的总归宿**。前面 7 个模块都会埋伏笔,但不剧透。如果你已经迫不及待,可以直接翻第 8 模块看看终点长什么样,再回头读 1–7。

## 这本书的偏见

我会坦白告诉你我的立场:

1. **数据 > 模型**。当模型和数据冲突时,先怀疑模型。
2. **机制 > 拟合**。Power law 拟合得好不算解释,必须有微观故事。
3. **承认局限**。市场不是物理系统,参与者会学、会变、会反向利用你的模型——这一点物理学家容易忘。
4. **数学够用即可**。没有形式化你说不清话,但形式化不是目的本身。**每个公式都先讲含义、再写形式、再讨论**。

## 数学深度

中等。本书不追求测度论级别的严谨,但会要求你能:

- 读懂 SDE,知道 Itô 引理在做什么
- 处理离散和连续时间随机过程的基本运算
- 看到 power law 知道怎么取 log-log 估指数
- 看到 Marchenko-Pastur 分布知道它在说协方差矩阵

如果某个数学工具暂时不会,每个模块的"延伸阅读"会给你指方向。

---

## 给读者的承诺

1. **不复读经济学教科书**。Hull 已经写得很好,这本书做的是它没做的事:从物理学家的世界出发翻译。
2. **每个模块配 Python lab**。不是 toy example,是直接用真实金融数据(S&P 500、加密货币 tick 数据等)。
3. **不藏复杂度**。重尾的拟合陷阱、RMT 去噪的边界、临界预警的争议 —— 该讲深的地方讲深,不掩盖权衡。
4. **不假装精通**。我自己卡过的地方会标记 "⚠️ 这里我也曾困惑",并说明怎么走出来。
